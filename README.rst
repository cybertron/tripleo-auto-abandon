===============================
tripleo-auto-abandon
===============================

Auto-abandon bot for the TripleO Gerrit projects.

Please feel here a long description which must be at least 3 lines wrapped on
80 cols, so that distribution package maintainers can use it in their packages.
Note that this is a hard requirement.

* Free software: Apache license
* Documentation: http://docs.openstack.org/developer/tripleo-auto-abandon
* Source: http://git.openstack.org/cgit/openstack/tripleo-auto-abandon
* Bugs: http://bugs.launchpad.net/tripleo
